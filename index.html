How to use sematic tags

Basic of html

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It describes the structure of a webpage using a series of elements and tags.

HTML elements are the building blocks of HTML pages. They are represented by tags, which are enclosed in angle brackets. For example, the <p> tag is used to define a paragraph.

What is a semantic tag?

Semantic tags are HTML tags that convey meaning about the content they contain. They help search engines and browsers understand the structure and purpose of the content on a webpage.

Examples of semantic tags include:

  • <header>: Represents the header section of a document or a section.
  • <nav>: Represents a navigation section.
  • <article>: Represents a self-contained piece of content.
  • <section>: Represents a thematic grouping of content.
  • <footer>: Represents the footer section of a document or a section.
